# Models

Unpack the Zenodo checkpoints here so these directories exist:

- `src/gap/models/gap_model/`
- `src/gap/models/sim-to-real_model/`

`gap_model/` is used by `RQ1` to `RQ5`.
`sim-to-real_model/` is used by `RQ6`.

These checkpoints are not stored in Git. The artifact expects the extracted
directories to contain the RLlib checkpoint trees exactly as shipped in the
Zenodo bundle.
